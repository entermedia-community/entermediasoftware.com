EnterMedia FAQ

 

Q. Can EnterMedia be setup and run in a multi-node environment with a load balancer?

 

A. Yes. We have two recommended configurations. The simplest is a single master multiple read only slaves. This allows each node to have an isolated copy of the metadata spread around the world using simple file replication. A more advanced configuration uses a distributed search that replicates edits in a bi-directional way and uses locks to maintain file integrity. This configuration assumes a shared file system. The load balancer will need to support sticky sessions so that the browser is routed to the same node for the life of the session.
