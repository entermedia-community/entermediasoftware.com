Latest release notes for EnterMedia Digital Asset Management Software are listed below.

 

June 2011 Release Notes for entermedia-server build 7.532

  • Updated and New EnterMedia applications such as EMTube, OpenEdit, EML and MediaLab applications available. 
  • Updated EnterMedia Server Administrative actions.
  • New Rhozet Carbon Copy integration for converting video assets.
  • New Aspera integration available for download acceleration.
  • New SmartJog integration available for broadcasters.
  • New FTP and SMB publish destinations for publishing converted assets automatically or when exporting.
  • New Asset Type support within a single catalog. Allows groups of metadata handling within a single catalog.
  • New User Profile support. Allows preferences and security levels to be defined on a per catalog basis.
  • Event Manager now uses sub-folders with groovy scripts for all major export and publish types.
  • Additional REST services and XML escapes. Hits per page is now maintained.

 

EnterMedia FAQ

 

1. What are the recommended hardware requirements for EnterMedia?

 

  • Windows: 2000 or higher
  • Linux - Ubuntu/Debian (Any flavor is fine, Ubuntu is our preference)
  • OS X
  • Indexing and virus protections turned off
  • Hardware: 4 Gig of Ram, 4 CPU's
  • Disk: Software needs 100 megs, 100,000 images needs 1-10 Gigs for thumbnails
  • Software: Latest versions of: Java, SSH, Tomcat, IPTables, Image Magick, EXIFTool, FFMPEG, LAME, Ghostscript

 

2. I need your SQL connector for my EnterMedia project, how do I get it?

 

The connector is in SVN. It is called entermedia-sql but is a bit out of date. We would need a couple of days work to get it up to date. If you would like to sponsor its development, we can send you a proposal. This allows us to dedicate a developer to speed up development. Otherwise, we will have an EnterMedia developer update it, where it will be part of a future release but we currently do not have a time line for this particular project. Please let us know if you are interested in sponsoring this EnterMedia project.
