We will officially announce our newest release of EnterMedia soon, until then, here's a sneak peek. You can download version 7.532 here http://entermediasoftware.com/download.html

  May 2011 entermedia-server build 7.532

  • EnterMedia server is now focused exclusively on Administrative actions.
  • EnterMedia applications such as EMTube, OpenEdit, EML and MediaLab applications available
  • Rhozet Carbon Copy integration for converting video assets
  • Aspera integration available for download acceleration
  • SmartJog integration available for broadcasters
  • FTP and SMB publish destinations for publishing converted assets automatically or when exporting
  • New Asset Type support within a single catalog. Allows groups of metadata handling within a single catalog
  • New User Profile support. Allows preferences and security levels to be defined on a per catalog basis
  • Event Manager now uses sub-folders with groovy scripts for all major export and publish types
  • Additional REST services and XML escapes. Hits per page is now maintained
